#Comandos do Git que precisa conhecer

1.  **cd** - navegar entre as pastas.
2.  **cd ..** - voltar um nível.
3.  **ls** - listar diretórios.
4.  **mkdir** - criar pasta.
5.  **git ini** - iniciar repositório.
6.  **git rm -rf .git** - apagar repositório.
7.  **git status** - verificar o status completo do repositório.
8.  **git clone** - para baixar o código-fonte existente de um repositório remoto, como a do GitHub.
9.  **git branch** - vários desenvolvedores podem trabalhar paralelamente no mesmo projeto.
10. **git push** - para subir (upar) a nova branch para o repositório remoto, como a do GitHub.
11. **git add** - adicionar uma alteração no diretório ativo.
12. **git commit** - para registrar as alterações em um arquivo no branch.
